# Challenge 2  - Intelligence Gathering

**Points:** 200  
**Protocol:** OPC-UA  
**Difficulty:** Beginner  

---

## The Story

You broke into the corporate portal and found a list of internal systems. One of them is a "data gateway"  - sounds important. Time to dig deeper into what the plant is hiding.

## What You Know (from Challenge 1)

- Admin panel showed several internal IPs
- There's a DMZ network at 10.0.2.x
- Something called "OPC-UA" is running somewhere

## Step 1: Scan the DMZ

The admin panel pointed to the DMZ network. Scan it to find what services are running.

**Hint:** Scan `10.0.2.0/24` with nmap.

<details>
<summary>Need the command?</summary>

```bash
nmap -sT 10.0.2.0/24
```
</details>

**What you'll notice:** One of the IPs from the admin panel doesn't show any open ports in the default scan. Why?

## Step 2: The Missing Service

Default nmap only scans the top 1000 ports. Industrial protocols use non-standard ports that get missed.

**Hint:** The admin panel told you exactly which port to look for. Scan it specifically.

<details>
<summary>What port?</summary>

OPC-UA uses port **4840**. Scan it directly:
```bash
nmap -sT -p 4840 10.0.2.30
```
</details>

## Step 3: Connect to OPC-UA

OPC-UA (Open Platform Communications Unified Architecture) is an industrial data protocol. It's like a database of everything happening in the plant.

**Hint:** You have a GUI tool installed. Try running `opcua-client` from the terminal.

<details>
<summary>How to connect?</summary>

1. Run `opcua-client` from a terminal
2. Enter the address: `opc.tcp://10.0.2.30:4840`
3. Click Connect
4. Browse the tree on the left panel
</details>

## Step 4: Explore the Data

You're now connected to the plant's data gateway. Browse through everything  - look for:
- Configuration data
- Credentials
- Hidden information
- Maintenance records

**Hint:** Expand every node. Some have sub-nodes that contain the real treasure.

<details>
<summary>Where to look specifically?</summary>

Look at these paths:
- `WaterTreatmentPlant → HistorianConfig`  - contains database credentials
- `WaterTreatmentPlant → PlantInfo → Maintenance → ServiceHistory`  - expand entries and check the Notes field
</details>

## Step 5: Write It Down

You found critical information. Write down:
- Any credentials (server, username, password)
- Any flags
- Any IP addresses and ports
- Network endpoints

**You'll need these for the next challenge.**

---

## What You Learned

- Industrial protocols use non-standard ports  - default scans miss them
- OPC-UA often allows anonymous access (no authentication)
- One connection can expose the entire plant's data
- Always scan specific ICS ports: 502, 4840, 20000, 44818, 2404, 47808
