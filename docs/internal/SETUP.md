# Setup

For the user-facing install walkthrough see [`../../INSTALL.md`](../../INSTALL.md). This document is an internal quick-reference for maintainers.

## Prerequisites

- Sysrupt board (4GB+ compute core)
- 32 GB+ microSD card
- Ethernet uplink (for install only)

## Install

```bash
git clone https://github.com/s34s0n/sysrupt-portable-ot-range.git
cd sysrupt-portable-ot-range
sudo ./install.sh
sudo reboot
```
